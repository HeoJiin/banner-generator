## 1. 경영 요약 (Executive Summary)

카카오스타일 파트너센터 배너 생성기는 **비디자이너(파트너 셀러, 운영자, 마케터)가 디자이너 도움 없이 광고 배너 4종을 직접 제작**할 수 있는 웹 기반 셀프서비스 도구입니다.

기존에는 배너 1건당 디자이너가 Figma에서 직접 제작해야 했으며, 평균 제작 소요 시간은 건당 30분~1시간, 수정 커뮤니케이션 포함 시 1~3일이었습니다. 본 도구 도입으로 **비디자이너가 5분 이내에 배너 4종을 동시 생성**할 수 있게 되어, 디자이너 리소스를 반복 업무에서 해방하고 배너 제작 병목을 해소합니다.

### 핵심 성과 요약

| 항목 | Before | After | 개선 효과 |
|------|--------|-------|-----------|
| 배너 제작 주체 | 디자이너 전담 | 누구나 셀프서비스 | 디자이너 의존도 탈피 |
| 배너 1건 제작 시간 | 30분~1시간 | **5분 이내** | **83~92% 단축** |
| 요청→완료 리드타임 | 1~3일 (대기 포함) | **즉시** | **99% 단축** |
| 브랜드 가이드라인 준수 | 제작자 역량에 의존 | 가이드 내 프리셋만 노출 | **100% 준수 보장** |
| 커버 가능 에셋 | 디자이너 재량 | 881개 3D + 100개 배경 | 확장 가능한 에셋 시스템 |

## 2. 문제 정의

### 2-1. As-Is 프로세스

```
배너 요청 발생 → 요청서 작성 → 디자이너 큐 대기 → Figma 제작 → 리뷰 → 수정 → 최종 내보내기 → 전달
```

- **평균 리드타임**: 1~3일 (디자이너 큐 대기 + 제작 + 수정)
- **순수 제작 공수**: 건당 30분~1시간 (4종 배너 기준)

### 2-2. 핵심 페인포인트

| # | 페인포인트 | 비즈니스 영향 |
|---|-----------|-------------|
| 1 | **디자이너 리소스 병목** | 시즌/캠페인 시기 배너 요청 폭증 → 디자이너가 핵심 프로젝트(신규 기능 디자인, UX 개선) 대신 배너 반복 작업에 투입 |
| 2 | **배너 제작 대기 시간** | 운영/마케팅팀의 프로모션 집행 지연 → 매출 기회 손실 |
| 3 | **품질 편차** | 제작자 숙련도에 따라 브랜드 가이드라인 준수 정도 상이 → 파트너센터 시각적 일관성 훼손 |
| 4 | **커뮤니케이션 비용** | 요청서 작성 → 리뷰 → 수정 요청의 왕복 커뮤니케이션 발생 |
| 5 | **확장 불가** | 배너 수요가 증가해도 디자이너 인력만큼만 처리 가능 |

### 2-3. To-Be 프로세스

```
배너 필요 → 배너 생성기 접속 → 키워드/시즌 선택 → 텍스트 입력 → 다운로드 → 파트너센터 적용
```

- **리드타임**: 즉시 (5분 이내)
- **디자이너 개입**: 불필요

## 3. 비즈니스 목표

### 3-1. 단기 목표 (0~3개월)

| 목표 | 성공 기준 | 측정 방법 |
|------|-----------|-----------|
| 디자이너 배너 제작 요청 70% 감소 | 월 배너 요청 건수 70% 이상 감소 | 디자이너 작업 요청 추적 |
| 셀프서비스 배너 월 50건 이상 | 비디자이너 다운로드 수 | 다운로드 이벤트 로그 |
| 배너 제작 5분 이내 | 평균 세션 시간 5분 이하 | 사용 시간 측정 |

### 3-2. 중기 목표 (3~6개월)

| 목표 | 성공 기준 |
|------|-----------|
| 배너 셀프서비스 비율 80% 이상 | 전체 배너 중 도구로 생성된 비율 |
| 파트너 셀러 자체 배너 제작 활성화 | 외부 사용자 월 20건 이상 |

### 3-3. 장기 목표 (6~12개월)

| 목표 | 성공 기준 |
|------|-----------|
| 디자이너 공수를 고부가가치 업무로 전환 | 디자이너 신규 기능 디자인 시간 30% 증가 |
| 배너 생성기를 플랫폼화 | 다른 서비스(톡채널 등)에도 배너 생성기 확장 |

## 5. 비즈니스 요구사항

### BR-01: 디자이너 리소스 병목 해소

| 항목 | 내용 |
|------|------|
| **문제** | 배너 제작이 디자이너에게 집중되어 핵심 프로젝트 진행 지연 |
| **요구사항** | 비디자이너가 브랜드 가이드라인 내에서 셀프서비스로 배너 제작 가능해야 함 |
| **해결책** | 소구점 키워드 기반 자동 테마 세팅 + 프리셋 옵션만 노출 |
| **구현 결과** | 5종 소구점 키워드 + 7종 시즌 강조 자동 세팅 시스템 구축 |

### BR-02: 배너 제작 속도 극대화

| 항목 | 내용 |
|------|------|
| **문제** | 배너 1건 제작에 30분~1시간, 요청 리드타임 1~3일 |
| **요구사항** | 배너 4종을 5분 이내에 생성하여 즉시 사용 가능해야 함 |
| **해결책** | 원클릭 4종 동시 생성 + 다시 뽑기 + ZIP 일괄 다운로드 |
| **구현 결과** | 키워드 1개 선택 → 배경/에셋/컬러 자동 세팅 → 4종 배너 즉시 프리뷰 → 2x PNG ZIP 다운로드 |

### BR-03: 브랜드 일관성 보장

| 항목 | 내용 |
|------|------|
| **문제** | 제작자에 따라 배너 품질과 가이드라인 준수 정도 편차 발생 |
| **요구사항** | 어떤 사용자가 만들어도 브랜드 가이드라인 범위 내 결과물이 나와야 함 |
| **해결책** | 디자이너가 검수한 컬러 팔레트/에셋만 프리셋으로 제공, 자유 입력 최소화 |
| **구현 결과** | 키워드별 60+ 컬러 베리에이션, 시즌별 25+ 전용 팔레트, 881개 검수된 3D 에셋 |

### BR-04: 확장 가능한 에셋 시스템

| 항목 | 내용 |
|------|------|
| **문제** | 새로운 시즌/캠페인마다 에셋을 코드에 하드코딩해야 하는 구조 |
| **요구사항** | 코드 수정 없이 에셋 추가만으로 확장 가능해야 함 |
| **해결책** | 파일명 기반 동적 에셋 로딩 시스템 (영어 접두사 기반 자동 분류) |
| **구현 결과** | `GET /api/assets` API가 디렉토리 스캔 → 파일명 토큰으로 키워드/시즌 자동 매핑 |

### BR-05: 다양한 배너 활용 시나리오 지원

| 항목 | 내용 |
|------|------|
| **문제** | 그래픽 에셋 기반 배너만으로는 상품 중심 프로모션 대응 불가 |
| **요구사항** | 실제 상품 사진을 활용한 배너 제작도 가능해야 함 |
| **해결책** | 듀얼 모드 (그래픽형 + 이미지형) 지원 |
| **구현 결과** | 이미지형 모드에서 사진 업로드 → 자동 색상 추출 → 그라데이션 오버레이 → 배너별 이미지 위치/크기 미세 조절 |

## 6. ROI 추정

> 아래 수치는 합리적 가정에 기반한 추정치입니다. 실제 운영 데이터로 검증이 필요합니다.

### 6-1. 가정

| 가정 항목 | 값 | 근거 |
|-----------|-----|------|
| 디자이너 시급 | 5만원 | 연봉 기준 역산 (시니어 디자이너 기준) |
| 월 배너 제작 요청 건수 | 40건 | 운영팀 요청 빈도 추정 |
| 건당 디자이너 순수 공수 | 45분 (0.75시간) | 4종 배너 제작 평균 |
| 건당 커뮤니케이션 비용 | 15분 (0.25시간) | 요청서 확인 + 리뷰 + 수정 피드백 |
| 도구 도입 후 셀프서비스 전환율 | 70% | 목표치 |
| 도구 개발 공수 | 80시간 | 프로젝트 실 투입 시간 |
| 개발자 시급 | 5만원 | 연봉 기준 역산 |

### 6-2. 월간 절감 효과

```
셀프서비스 전환 건수 = 40건 x 70% = 28건/월

디자이너 공수 절감
  = 28건 x (0.75 + 0.25)시간 x 5만원
  = 28 x 1시간 x 5만원
  = 1,400만원/월 (시간 가치 환산)

디자이너 절감 시간
  = 28건 x 1시간 = 28시간/월
```

### 6-3. 연간 절감 효과

```
연간 공수 절감 = 1,400만원 x 12 = 1억 6,800만원/년 (시간 가치 환산)
연간 절감 시간 = 28시간 x 12 = 336시간/년 = 약 42 인일/년
```

### 6-4. 투자 회수 기간 (Payback Period)

```
개발 투자 비용 = 80시간 x 5만원 = 400만원

투자 회수 기간 = 400만원 / 1,400만원 = 약 0.3개월 (약 9일)
```

### 6-5. 부가 가치 (정량화 어려운 효과)

| 항목 | 효과 |
|------|------|
| **프로모션 대응 속도** | 배너 제작 리드타임 1~3일 → 즉시, 매출 기회 포착 속도 향상 |
| **디자이너 업무 질 향상** | 반복 배너 대신 신규 기능 디자인/UX 개선에 집중 가능 |
| **운영팀 자율성** | 디자이너 일정에 의존하지 않는 독립적 배너 운영 |
| **브랜드 일관성** | 프리셋 기반으로 모든 배너의 품질 균일화 |
| **에셋 자산 축적** | 881개 3D 에셋 + 100개 배경 = 재사용 가능한 디자인 자산 |

### 9-2. 핵심 기술 차별점

1. **파일명 기반 동적 에셋 시스템**: 코드 수정 없이 파일 추가만으로 881개 이상의 에셋 관리. 영어 접두사 기반 자동 분류로 키워드x시즌 조합별 에셋 매핑 자동화.

2. **전용 디자인 시스템 구축**: 5카테고리 시맨틱 컬러 토큰 + 17개 재사용 UI 컴포넌트 + `/tokens` 시각화 페이지. 배너 생성기 고유의 UI 언어 확립.

3. **듀얼 모드 아키텍처**: 그래픽형(3D 에셋)과 이미지형(실사 이미지)을 동일한 템플릿 시스템으로 처리. 이미지 업로드 시 Canvas API로 자동 색상 추출 → 배경/텍스트 대비 자동 설정.

4. **인스턴스 복제 시스템**: 배너 인스턴스 단위로 설정을 상속하되 개별 오버라이드 가능. 동일 캠페인 내 문구 베리에이션을 효율적으로 생성.

5. **230+ 컬러 베리에이션**: 키워드별 60+, 시즌별 25+ 컬러 팔레트. 가중치 기반 랜덤 선택으로 매번 다른 조합을 제공하면서도 브랜드 일관성 유지.
